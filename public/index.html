// FAQ
What metadata does PrivacyClean remove?
For images: EXIF data (camera model, GPS coordinates, date/time, software used), IPTC captions, and XMP metadata. For PDFs: Author, Creator, Producer, Title, Subject, Keywords, Creation/Modification dates, and XMP metadata streams.
Is my data really private?
All file processing happens in your browser using WebAssembly and Web APIs. No file data is ever transmitted to our servers. You can verify this by checking your browser's Network tab — you won't see any file uploads.
Will image quality be affected?
PrivacyClean uses WASM-powered lossless stripping — we remove only the metadata segments, not the image data. Your photos remain pixel-for-pixel identical. No re-encoding, no compression, no quality loss. If WASM is unavailable, we fall back to Canvas re-encoding at 92% JPEG quality.
Does it work on mobile?
Yes. PrivacyClean works on any modern browser including iOS Safari and Android Chrome. The experience is fully responsive and optimized for touch screens.
Can I get a refund?
We offer a 30-day money-back guarantee on all paid plans. If you're not satisfied, contact us within 30 days for a full refund — no questions asked.
Is this GDPR and CCPA compliant?
Yes. We follow privacy by design principles: no analytics, no tracking cookies, minimal data collection. We only store your email for authentication and subscription status. Files are never uploaded. See our Privacy Policy for details.
What's the difference between Pro and Lifetime?
Both unlock the same features. Pro is a monthly ($4/mo) or annual ($39/yr) subscription. Lifetime is a one-time payment of $89 — pay once and use forever with all future updates included. Lifetime is the best value if you plan to use PrivacyClean regularly.
