// FAQ
What metadata does a PDF contain?
PDFs can contain Author, Creator (app used to create the PDF), Producer (PDF library), Title, Subject, Keywords, Creation Date, Modification Date, and XMP metadata streams. Many of these are auto-populated by your software.
Can PDF metadata reveal personal information?
Yes. The Author field often contains your real name from your OS account. The Creator field reveals what software you used. Creation/Modification dates show your working patterns and last-minute edits.
Does stripping PDF metadata affect the content?
No. PrivacyClean only removes metadata fields (Author, Creator, etc.) and XMP streams. The document text, images, formatting, and layout remain exactly the same. The content is completely unchanged.
How do I check PDF metadata?
In any PDF reader: File → Properties (or Ctrl/Cmd+D). You'll see Author, Title, Creation Date, and more. Anyone who receives your PDF can see this information.
